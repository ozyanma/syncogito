# ブラウザ探索型ARG：失われた歌声（前半パート）

本リポジトリは、架空の企業のホームページを探索してストーリーを読み解いていく、ブラウザ探索型ARGのソースコード管理領域です。
※多忙のため、後半パートの公開まで時間かかりそうということで前半だけ先出ししてます…

## 🌐 プレイ用URL（GitHub Pages）
ゲームのプレイは以下のリンクから行ってください。
https://ozyanma.github.io/syncogito/index.html

---

## 📝 ゲームの概要
プレイヤーはポッドキャスト『ひまみなもの好き放題ラジオ』パーソナリティの「まじめちゃん」となり、突然「助けて」というチャットを送ってきた相方「ゆりかっぺ」を救うため、彼女が案内してきた不審な企業「シン・コギト株式会社」のホームページを調査します。
捕われたゆりかっぺとチャットしつつ、サイト内に隠された様々な手がかりからキーワードを連想し、検索窓に入力していくことでストーリーを進めていくゲームです。

---

## 🔍 遊び方とルール

### 1. 検索キーワードの探索
ゲームを進めるためには、サイト内のテキストを細かく読み、**怪しい単語（キーワード）を推測して検索窓に入力**する必要があります。

### 2. 検索窓の入力仕様
* システムが認識するキーワードは、基本的に**「1単語」**またはスペースで区切られた**「2単語（アンド検索）」**です。
* 長い文章や文章形式で入力してもヒットしません。さらに情報を絞り込みたい場合は、2つの単語を組み合わせて検索してください。

### 3. ソースコード閲覧（ネタバレ）に関する注意
本ゲームはフロントエンドの技術のみで動作しているため、リポジトリ内のファイルを直接閲覧したり、ブラウザの開発者ツール（F12キー）を動かしたりすると、**隠しページのURLや検索キーワード、今後のストーリー展開がすべて見えてしまいます。**
ゲーム本来のミステリーや謎解きを楽しんでいただくため、**ソースコードの直接閲覧や解析は行わずにプレイすることを強く推奨します。**
（URLが連番になっているので、どうしても詰まったら直接URLいじってください）
